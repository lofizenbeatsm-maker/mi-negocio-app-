Sistema de Administración para Negocio Mixto

1. Descripción General

Esta aplicación web/móvil ha sido diseñada específicamente para administrar un negocio de comercio minorista que opera tres categorías de productos: Charcutería, Víveres y Papelería.

La aplicación incluye:

  • ✅ Sistema de Punto de Venta (POS) completo
  • ✅ Gestión de Inventario en tiempo real
  • ✅ Control de usuarios por roles (Admin y Empleado)
  • ✅ Reportes de ventas y ganancias
  • ✅ Gestión de clientes y deudas
  • ✅ Administración de empleados
  • ✅ Alertas de productos vencidos y bajo stock
  • ✅ Funcionamiento sin conexión a internet (con sincronización posterior)

3. Módulos Principales

📊 Panel Principal (Dashboard)

Funcionalidad:
  • Resumen de productos por categoría (Charcutería, Víveres, Papelería)
  • Ventas totales del día
  • Listado de productos con bajo stock (≤20 unidades)
  • Alertas visuales para reorden

🛒 Punto de Venta (POS)

Características:
  • Búsqueda rápida de productos por nombre
  • Filtrado por categoría (Charcutería, Víveres, Papelería)
  • Carrito dinámico con vista previa de total
  • Ajuste de cantidades en tiempo real
  • Múltiples métodos de pago: Efectivo, Tarjeta, Mixto
  • Recibos digitales (preparado para impresión)
  • Cierre de turno (apertura/cierre de caja)

Cómo usar:

  1. Busca un producto por nombre o navega por categoría
  2. Haz clic en el producto para agregarlo al carrito
  3. Ajusta cantidades directamente en el carrito
  4. Selecciona método de pago
  5. Haz clic en "Completar Venta" para registrar la transacción

📦 Inventario

Funcionalidad (Admin solo):
  • Visualización de todos los productos
  • Agregar nuevos productos con nombre, categoría, precio, costo
  • Monitoreo de stock disponible
  • Registro de fecha de vencimiento (para charcutería)
  • Cálculo automático de ganancia por producto
  • Alertas de productos próximos a vencer (fondo rojo)
  • Eliminación de productos

Campos de un Producto:

  • Nombre: Identificación del producto
  • Categoría: Charcutería, Víveres o Papelería
  • Precio: Precio de venta al público
  • Costo: Costo de compra al proveedor
  • Cantidad: Stock disponible
  • Fecha de Vencimiento: Fecha de expiración (opcional, recomendado para charcutería)

👥 Gestión de Clientes

Funcionalidad (Admin solo):
  • Registro de clientes frecuentes
  • Almacenamiento de teléfono de contacto
  • Control de deudas pendientes (crédito)
  • Historial de compras (visible en reportes)
  • Identificación visual de clientes con deuda

👔 Administración de Empleados

Funcionalidad (Admin solo):
  • Registro de empleados con nombre y usuario
  • Asignación de roles: Admin o Empleado
  • Control de permisos por rol
  • Seguimiento de ventas por empleado (en reportes)

📈 Reportes y Analytics

Reportes disponibles:
  • Venta total acumulada
  • Ganancia potencial (basada en costo vs precio)
  • Ventas por categoría (Charcutería, Víveres, Papelería)
  • Historial completo de transacciones
  • Detalles de cada venta (items, total, método de pago)
  • Filtrado por fecha y empleado

6. Funcionalidades Principales Explicadas

6.1 - Sistema de Inventario

El inventario es el corazón de la aplicación:

  • Actualización en tiempo real: Al completar una venta, el stock se reduce automáticamente
  • Alertas de bajo stock: Productos con ≤20 unidades aparecen resaltados en rojo
  • Control de vencimiento: Especialmente importante para charcutería
  • Cálculo de ganancia: (Precio - Costo) × Cantidad = Ganancia potencial

6.2 - Sistema POS

El Punto de Venta optimizado para velocidad:

  • Búsqueda instantánea: Escribe el nombre del producto y aparece inmediatamente
  • Carrito editable: Modifica cantidades sobre la marcha
  • Múltiples métodos de pago: Efectivo, tarjeta o mixto
  • Recibos digitales: Cada venta genera un recibo que puede imprimirse o enviarse

6.3 - Reportes y Analítica

Datos clave para tomar decisiones:

  • Venta total: Ingresos acumulados
  • Ganancia: Basada en la diferencia entre costo y precio de venta
  • Ventas por categoría: Identifica qué sección genera más ingresos
  • Historial: Consulta cualquier transacción pasada con detalles completos

8. Personalización y Configuración

Cambiar Productos Iniciales

Para modificar los productos que vienen precargados, edita la sección de datos en el código HTML:

const [inventory, setInventory] = useState([ { id: 1, name: 'Jamón Serrano', category: 'charcuteria', price: 25000, cost: 15000, quantity: 20, expiryDate: '2024-12-31' }, // Agrega más productos aquí... ]);

Cambiar Información de la Empresa

Busca la línea en el código:

<div class="logo">🏪 Mi Negocio</div>

Reemplaza "Mi Negocio" con el nombre real de tu empresa.

Cambiar Colores y Estilos

Los colores principales se encuentran en la sección de estilos CSS. Los colores clave son:

  • #1e3c72 - Azul oscuro (encabezados)
  • #2196F3 - Azul claro (acentos)
  • #4CAF50 - Verde (éxito)
  • #ff6b6b - Rojo (peligro)

10. Soporte y Mantenimiento

Problemas Comunes

Problema Solución
Datos se pierden al cerrar navegador Los datos se guardan solo en la sesión actual. Para persistencia, se necesita base de datos remota.
No puedo ver los productos agregados Actualiza el navegador (F5). Los datos están en memoria de React.
El carrito no se actualiza Cierra el modal si está abierto. A veces hay conflicto con formularios.
Contraseña olvidada En demo es siempre "123". En producción, implementar recuperación por email.
