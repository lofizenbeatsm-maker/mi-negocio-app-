⚠️ Importante: Próximas Mejoras Recomendadas

La versión actual es MVP (Minimum Viable Product) - funciona perfectamente pero puede mejorarse. Se recomienda:

  • Base de datos remota: Para guardar datos permanentemente (Firebase, MongoDB)
  • Sincronización automática: Cuando hay internet, los datos se sincronizan
  • Impresión integrada: Lectura de códigos de barras, impresoras Bluetooth
  • Integración de pagos: Conexión con procesadores de tarjeta
  • Tienda online: Catálogo virtual para vender por WhatsApp/web
  • Envío de recibos: Automático por WhatsApp, email o SMS

🔐 Preguntas Importantes Respondidas

¿Necesito internet para usar la app?

No. La app funciona 100% sin internet. Tu dispositivo funciona de forma autónoma. Cuando haya internet (versión mejorada), se sincronizará automáticamente a la nube.

¿Qué pasa si pierdo mi teléfono/tablet?

En la versión actual, los datos se pierden. En la versión mejorada con base de datos remota, tus datos estarán seguros en la nube y podrás acceder desde cualquier dispositivo.

¿Puedo usar esto en múltiples locales?

Versión actual: No. Cada instalación es independiente. En la versión mejorada podrás sincronizar múltiples sucursales desde un panel central.

¿Qué tan segura es mi información?

Los datos se guardan localmente en tu dispositivo. Solo tú accedes con tu usuario y contraseña. En la versión mejorada, habrá encriptación de end-to-end para máxima seguridad.

¿Cuánto cuesta mantener esto?

Versión actual (MVP): GRATIS. La inversión fue única en desarrollo (USD $3-5K). El mantenimiento anual sería USD $500-1,000 si decidimos agregar base de datos remota.
